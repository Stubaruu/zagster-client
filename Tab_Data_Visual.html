The Data above labled as Hours, shows the number of rides per hour. As you can see, the 16th hour of the day (or 4pm) is the most popular time of day

The Data above labled as Years, shows the total number of rides per year. As you can see, 2018 has been the most popular year.

About

The graphs shown above are a combination of bringing in data via JSON requests from a herokuapp service. The data is then displayed displayed using a $.when(...).then(...) command. The Javascript is pulled into the HTML file using a canvas element, which then displays the graphs.
